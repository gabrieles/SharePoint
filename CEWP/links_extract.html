This page takes 5 parameters from the URL:

  • listname: the full name of the list you want to extract the links from (eg: "Tasks")
  • fieldname: the name of the field that you want to scan for links (eg: "Task Content" or "Description")
  • listContext: the relative URL of where the list is (eg: "/" or "/directorates/Communications/")
  • highlight1: all URLs with this string will be highlighted with a yellow background (eg: "ispace.icr.ac.uk")
  • highlight2: all URLs with this string will be highlighted with a green background (eg: "cspace.icr.ac.uk")

NB: they are all case sensitive. You can leave spaces in the fieldName.

Examples

  • Tasks
  • Our ICR
  • Policy library
